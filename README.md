## Описание задания

---

## Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd lab
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

Содержимое проекта:

 ```docker-compose.yml ``` — файл конфигурации Docker Compose для запуска контейнеров PHP, Redis, Elasticsearch и ClickHouse

 ```Dockerfile ``` — файл сборки контейнера PHP с установленным Composer и библиотекой Guzzle

 ```www/index.php ``` — основная страница приложения для запуска примеров работы с нереляционными базами данных

 ```www/RedisExample.php ``` — класс для взаимодействия с Redis через HTTP API с использованием Guzzle

 ```www/ElasticExample.php ``` — класс для работы с Elasticsearch (добавление и поиск данных) через REST API

 ```www/ClickhouseExample.php ``` — класс для выполнения SQL-запросов к ClickHouse через HTTP API

 ```www/ClientFactory.php ``` — вспомогательный класс для создания HTTP-клиента Guzzle

 ```www/composer.json ``` — файл конфигурации Composer с зависимостью библиотеки Guzzle

 ```www/db.php``` — файл подключения и настройки клиента для работы с сервисами

 ```www/vendor/  ```— папка с установленными библиотеками Composer

 ```composer.lock ``` — файл фиксации версий установленных зависимостей проекта
